Behavior Designer

Introduction#

BehaviorDesigner はノードを組むことで、AIの判断と行動をとらせる手法およびアセットです。 Recollect では独自Taskで判断・行動を追加しており、A* Pathfinding Project Pro と組み合わせることで、キャラクターの移動を実現しています。

詳細は RecollectActor の AI ページ を参照してください。

設定ファイルについて#

Project ウィンドウで右クリックし Create > Behavior Designer > External Behavior Tree で設定ファイルを作成できます。 あるいはツールバーの Assets > Create > Behavior Designer > External Behavior Tree でも可能です。

ノード操作画面について#

BehaviorDesigner の設定ファイル（External Behavior Tree）の内容を変更するには、専用のウィンドウを開く必要があります。 ツールバーの Tools > Behavior Designer > Editor から開くことができます。

BehaviorDesigner の基本ノード#

BehaviorDesigner はノードを組むことで、AIの判断と行動をとらせる手法およびアセットです。 よく利用するノードをピックアップして解説します。

Composites#

ノードにどういった方法でアクセスするかを指定します。

Decorators#

ノードの状態（待機、成功、失敗など）によって条件分岐します。

Actions#

何か行動を起こし、ノードの状態（待機、成功、失敗など）や変数を変化させます。

Conditionals#

GameObject が存在するかなど、Decorators よりも細かい単位で条件判定を行い、ノードの状態（待機、成功、失敗など）を変化させます。
