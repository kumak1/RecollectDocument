A*

Introduction#

A* Pathfinding Project Pro は高速で経路探索を行うアセットです。 Recollect では BehaviorDesigner の 独自Task 内で利用しており、経路探索結果をコントローラーによる入力操作に変換し、キャラクターの移動操作をおこなうことができます。

詳細は RecollectActor の AI ページ を参照してください。
