Barricade

概要​

普段は非表示だが Collider に触れたら表示に切り替える、壁用のコンポーネント BarricadeSetUpper です。 Recollect Shader を利用している場合は Fade In/Out に対応し、それ以外の場合は即時表示・非表示が切り替わります。

Property:
Barricade Mesh表示・非表示する GameObject を指定します
Collider Mesh表示・非表示の切り替えに使う Collider を指定の Mesh から生成します
Shape TypeCollider Mesh で指定した Mesh から生成する Collider の形を選択します
Fit TypeCollider を Mesh の内側に作成するか、外側に作成するか選択します
ConvexonにするとColliderのポリゴン数は抑えられますが、意図しない形になる可能性があります
Fade SpeedRecollect Shader を利用している場合の Fade In/Out 速度を指定します
Fade CurveFade In/Out のアニメーションカーブを指定します

Hierarchy への反映​

設定更新する​

Process ボタンを押下すると、以下の挙動をします。

  • Barricade Mesh Collider Mesh で設定した GameObject に対して、Barricade として必要な Component や設定を追加します。

設定を取り除く​

Cleanup ボタンを押下すると、以下の挙動をします。

  • Process で追加した Component を削除します
