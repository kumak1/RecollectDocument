RecollectUiEvent

概要​

GUI のクリックイベントに、Recollect独自のイベントを追加します。

Property:
Mode動作するイベントの種類を指定します。

Mode 毎の設定値について​

Situation​

Property:
Director切替方法を Slider, String から選択します。
Situation IndexSituation を数値で指定します。(Director: Sliderのみ)
Shot IndexShot を数値で指定します。(Director: Sliderのみ)
Shot NameShot を文字列で指定します。(Director: Stringのみ)
Control
Enable Rotateクリック後にカメラを回転できるか指定します。

Spawn​

Property:
Director
SpawnActor切り替える SpawnParam を指定します。
Player
ProfileSpawn する Player を上書きしたい場合、設定ファイルを指定します。
MotionPlayer の Motion を上書きしたい場合、設定ファイルを指定します。
CameraAdjustPlayer に紐づく CameraAdjust を上書きしたい場合、設定ファイルを指定します。
EquipSetPlayer の EquipSet を上書きする方法を、None, Toggle, Specify, Optional から選択します。
KeepOptionaltrue の場合、 is Optional の表示・非表示は維持されます。false の場合は is Optional が全て非表示になります。
PrefixEquipSetの 1_0_0 の 1 が共通としたの別 EquipSet に切り替えます。この 1 を切り替える 8 などを指定します。
空欄の場合は現在の EquipSet の左の文字を共通とした別のEquipSetに切り替えます。(EquipSet: Toggle のみ)
NamesPlayer で上書きする設定を文字列で指定します。 (EquipSet: Specify のみ)
OptionalModeEquipSet のうち Optional だけの操作を行います。Toggle, Show, Hide から選択します。
EquipSetNameOptional の操作する EquipSet を指定します。
AfterActionクリック後に Player で実行する Motion を指定します。
Control
Enable Rotateクリック後にカメラを回転できるか指定します。
UI
TextRefreshRecollectUiTextInjector を上書きする方法を、None, Toggle, Specify から選択します。
TargetsRecollectUiTextInjector を文字列で指定します。(TextRefresh: Specify のみ)
ModalModal の方法を、None, Open, Close から選択します。
Name開く Modal の Path を文字列で指定します。(Modal: Open のみ)
AnimationModal を Open/Close する際に Animation を有効にするか指定します。

Player

MotionPlayer の Motion を上書きしたい場合、設定ファイルを指定します。

空欄の場合は現在の EquipSet の左の文字を共通とした別のEquipSetに切り替えます。(EquipSet: Toggle のみ)
NamesPlayer で上書きする設定を文字列で指定します。 (EquipSet: Specify のみ)
AfterActionクリック後に Player で実行する Motion を指定します。
Control
Enable Rotateクリック後にカメラを回転できるか指定します。
UI
TextRefreshRecollectUiTextInjector を上書きする方法を、None, Toggle, Specify から選択します。
TargetsRecollectUiTextInjector を文字列で指定します。(TextRefresh: Specify のみ)
