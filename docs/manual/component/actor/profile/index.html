Profile

概要​

FBX​

キャラクターのメッシュが記録されているFBXファイルを指定します。

Description​

DescriptionParam でキャラクターの詳細情報をメモすることができます。

  • Character Name
    • キャラクターの名前を文字列でメモします

AI​

Actor Type​

キャラクターのタイプ（他キャラクターのAIにどう認識されるか）を指定します。

  • Playable
    • 操作キャラクター
  • NpcTalker
    • 喋ることができるNPC
  • NpcStore
    • イベント（ショップ）のあるNPC
  • NpcEnemy
    • 敵となるNPC

Behavior Tree​

BehaviorDesigner の 設定ファイル（External Behavior Tree） を指定します。
