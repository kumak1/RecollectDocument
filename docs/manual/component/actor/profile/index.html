FBX​

キャラクターのメッシュが記録されているFBXファイルを指定します。

Description​

DescriptionParam でキャラクターの詳細情報をメモすることができます。

  • Character Name
    • キャラクターの名前を文字列でメモします

Actor Type​

キャラクターのタイプ（他キャラクターのAIにどう認識されるか）を指定します。

NpcEnemy敵となるNPC
