FBX Importer

概要#

UnityではFBXファイルを Inspector で確認すると Animation タブでアニメーションのインポート設定を変更できたり、 Clips にて指定範囲で切り出して利用することができます。

一見便利なのですが、設定値はFBXのmetaファイルに保存されており、ファイルの更新・名称変更などに弱い管理方法となっています。 これを解決するために FBX Importer を開発・用意しています。

JSONファイルの配置方法#

FBX Importer は FBXファイルのImport時に自動的に適用されます。 FBXファイルと同名のJSONファイルを用意することで、JSONで指定した設定を自動的にFBXファイルに適用します。

JSONファイルの作成#

JSON ファイルは以下のように記述します。

{
"FbxAnimationCompression" : "Reduction",
"FbxAnimationResampleCurves" : true,
"FbxAnimationRotationError" : 0.5,
"FbxAnimationPositionError" : 0.5,
"FbxAnimationScaleError" : 0.5,
"FbxAnimationClips": [
{
"name": "Striker_Combine_DoublePunch",
"firstFrame": 3,
"lastFrame": 33,
"loop": "",
"FbxAnimationEvents": [
{ "functionName": "EventMove", "frame": 3, "MoveDistance": 0.0 }
]
}
]
}
Property:Function:
FbxAnimationCompressionAnim. Compression の設定を行います。reduction optimal off を指定できます。
FbxAnimationResampleCurvesResample Curves の設定を行います。 true false を指定できます。
FbxAnimationRotationErrorRotation Error の設定を行います。 float で値を指定できます。
FbxAnimationPositionErrorPosition Error の設定を行います。 float で値を指定できます。
FbxAnimationScaleErrorScale Error の設定を行います。 float で値を指定できます。
FbxAnimationClipsClips の分割指定を行います。 別途詳細を記述しています。

{
// 中略

FbxAnimationEvents の詳細#

RecollectActor にはキャラクターを操作する様々な public function が用意されており、 Animation Clip の Events に設定することで利用できます。 FbxAnimationEvents に設定を書き加えることで、

共通パラメータ#

Property:Function:
functionName動作させたい関数名を指定します。
frameFbxAnimationClips の firstFrame/lastFrame で切り出した後のフレーム数を指定します。アニメーション全体から割り出したフレーム数ではないので注意してください。

EventMove#

キャラクターを指定方向に移動します。

{
"functionName": "EventMove",
"frame": 3,
"MoveDistance": 0.0
}
Property:Function:
MoveDistance移動する距離を float で指定します。

EventStep#

キャラクターを指定方向に瞬間的に移動します。

{
"functionName": "EventStep",
"frame": 3,
"MoveDistance": 0.0
}
Property:Function:
MoveDistance移動する距離を float で指定します。

EventAbsorb#

周辺のキャラクターを指定座標に集めます。

{
"functionName": "EventAbsorb",
"frame": 3,
"AbsorbOffset": {"x": 0.0,"y": 0.0,"z": 0.0},
"AbsorbRange": 3.0,
"AbsorbAmount": 1.0
}
Property:Function:
AbsorbOffset周辺キャラを集める座標を Vector3 で指定します。(0,0,0) でキャラの原点です。
AbsorbRange周辺キャラを集める円の大きさを float で指定します。
AbsorbAmount周辺キャラを集める強さを float で指定します。

EventRotate#

キャラクターを指定方向に瞬間的に回転します。

{
"functionName": "EventRotate",
"frame": 3,
"RotateRight": 15,
"RotateLeft": 0
}
Property:Function:
RotateRight回転する角度（degree）を float で指定します。
RotateLeft回転する角度（degree）を float で指定します。

EventAttack#

キャラクターを指定方向に瞬間的に回転します。

{
"functionName": "EventAttack",
"frame": 4.5,
"AttackColliderOffset": {"x": 0.0, "y": 0.0, "z": 0.0},
"AttackColliderRadius": 0.0,
"BlowDirection": "Front",
"BlowDistance": 0.05,
"DamageMotionType": "LightFrontLeft"
}
Property:Function:
AttackColliderOffset攻撃ヒット判定用コライダーの座標を Vector3 で指定します。(0,0,0) でキャラの原点です。
AttackColliderRadius攻撃ヒット判定用コライダーの半径を float で指定します。
BlowDirection攻撃ヒット時に吹き飛ばす方向を Front Back Right Left から指定します。
BlowDistance攻撃ヒット時に吹き飛ばす距離を float で指定します。
DamageMotionType攻撃がヒットした際のダメージモーションを文字列で指定します。

DamageMotionType に指定する文字列#

Light
LightFrontLeft
LightFrontRight

HeavyFrontRight

BlowDodge

FatalBlowDodge

FatalFlyDodge

EventSuperArmorOn#

キャラクターのスーパーアーマーを指定の強さでOnにする。

{
"functionName": "EventSuperArmorOn",
"frame": 4.5,
"Tier": 0
}
Property:Function:
Tierスーパーアーマーの強さを 0 ~ 3 で指定します。

EventSuperArmorOff#

キャラクターのスーパーアーマーを解除します。

{
"functionName": "EventSuperArmorOff",
"frame": 4.5
}

EventCounterOn#

キャラクターのカウンターを指定の強さでOnにする。

{
"functionName": "EventCounterOn",
"frame": 4.5,
"Tier": 0
}
Property:Function:
Tierカウンターの強さを 0 ~ 3 で指定します。

EventCounterOff#

キャラクターのカウンターを解除します。

{
"functionName": "EventCounterOff",
"frame": 4.5
}

EventAttackBreakOn#

キャラクターの攻撃を相殺（or パリィ）します。

{
"functionName": "EventAttackBreakOff",
"frame": 4.5,
"Tier": 0
}
Property:Function:
Tier相殺の強さを 0 ~ 3 で指定します。

EventAttackBreakOff#

キャラクターの攻撃を相殺（or パリィ）します。

{
"functionName": "EventAttackBreakOff",
"frame": 4.5
}

EventLockOnBreak#

キャラクターのロックオン状態を解除します。

{
"functionName": "EventLockOnBreak",
"frame": 4.5
}
