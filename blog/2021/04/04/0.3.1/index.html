Recollect 0.3.1

April 4, 2021 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2021.0404.1.script.unitypackage

変更箇所#

  • Recollect Director
    • Audio Type
      • FMOD を無効化しました
  • その他
    • Behavior Designer を 1.6.8 にバージョンアップしました
    • FMOD を削除しました

今後修正すること#

  • Recollect Actor
    • Lock On
      • Lock On Move の場合、前回入力とは真逆の方向に移動する場合は速度が落ちるようになりました
    • Turn After Dodge にも ToBasic と専用のアニメーション再生用の項目を用意する
      • 通常時とバトル時で項目分けよう
    • Turn後の攻撃で移動パラメータがリセットされてなさそう
    • Damage Equip Sets
      • ダメージ状態によって、Material と BlendShape を変更できるようにする
    • Running
      • Start / End のモーション設定追加
    • その他
      • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
  • Recollect Director
    • 背景をうつすと処理負荷が高い原因を探る
    • その他
      • SceneView で PostEffect の On/Off を切り替えられるようにする
      • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
      • コマ止めのアニメ表現の実現方法を調べる
  • Level Design
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Shader
    • キャラクターにカラーパレット設定を追加
  • Config
    • キーコンフィグ画面の作成
