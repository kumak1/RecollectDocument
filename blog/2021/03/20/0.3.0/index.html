Recollect 0.3.0

March 20, 2021 · One min read
kumak1
Director, Chief Engineer

ダウンロード#

Recollect.2021.0321.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Turn
      • PlugableAnimeOffence に設定値を追加しました
      • Turn後につながる攻撃を Basic1st か AfterTurn1st どちらするか選べるようになりました
      • Turn の途中で Dudge ができるようになりました。
    • Equip Items
      • enable/disable 時に Materialize FX を設定できるようになりました
    • Sound
      • サウンドエンジンとして FMOD を採用しました。
      • Event EventAudio を追加し、アニメ再生中に音を再生できるようにしました。
        • 設定値の詳細は ドキュメント を確認ください。

今後修正すること#

  • Recollect Actor
    • Damage Equip Sets
      • ダメージ状態によって、Material と BlendShape を変更できるようにする
    • Running
      • Start / End のモーション設定追加
    • その他
      • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
  • Recollect Director
    • 背景をうつすと処理負荷が高い原因を探る
    • その他
      • SceneView で PostEffect の On/Off を切り替えられるようにする
      • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
      • コマ止めのアニメ表現の実現方法を調べる
  • Level Design
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Shader
    • キャラクターにカラーパレット設定を追加
  • Config
    • キーコンフィグ画面の作成
