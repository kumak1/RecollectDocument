Recollect 0.3.2

May 7, 2021 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

  • Recollect.2021.0507.1.script.unitypackage
  • Recollect.2021.0507.1.offence.unitypackage

変更箇所#

  • Recollect Actor
    • Lock On
      • Lock On Move の場合、前回入力とは真逆の方向に移動する場合は速度が落ちるようになりました
        • Reverse Dir Attenuation > Tolerance などで調整できます
    • Turn後の Dodge にも Basic か専用設定を選択できるようになりました
    • 通常時とバトル時で設定を変更できるように、Dodge1, Dodge2 を設定できるようになりました
    • Turn後の攻撃で移動パラメータがリセットされていない問題を解決しました
  • その他
    • Rewired を 1.1.39.1 にバージョンアップしました
    • Tail Animator を 2.0.5 にバージョンアップしました
    • Spine Animator を 2.0.0 にバージョンアップしました
    • Look Animator を 2.0.1 にバージョンアップしました

今後修正すること#

  • Recollect Actor
    • Lock on の範囲からはなれ、解除時に手が震える
    • Damage Equip Sets
      • ダメージ状態によって、Material と BlendShape を変更できるものを InOut Equip Sets とは別に用意する
      • 真っ二つ
        • https://www.youtube.com/watch?v=i--UUeGTMvo
    • Running
      • Start / End のモーション設定追加
    • その他
      • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
  • Recollect Director
    • Bloom を初回表示時に適用されない
    • 背景をうつすと処理負荷が高い原因を探る
    • その他
      • SceneView で PostEffect の On/Off を切り替えられるようにする
      • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
      • コマ止めのアニメ表現の実現方法を調べる
  • Level Design
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Shader
    • キャラクターにカラーパレット設定を追加
    • ColorMap の Enum EIGHT -> SEVEN
  • Sound
    • FMODに変わる音再生システムの構築
  • Config
    • キーコンフィグ画面の作成
