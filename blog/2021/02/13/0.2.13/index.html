Recollect 0.2.13

February 13, 2021 · One min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2021.0213.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Mask
      • SkinSwitch の Mask テクスチャが正しく動作していない問題を解決しました
    • BlendShape
      • Overwrite Object > BlendShapes で Skin Switch Param の BlendShape を上書きできるようになりました。
        • 後述の共通で利用する Equip Items の Blend Shape の上書きなどに利用してください。
    • Equip Items
      • SkinSwitch を従来のものと別途設定できるようになりました。
      • Equip Items の SkinSwitch では Default Equip Sets のチェックがされたものだけ表示されます。
      • InOut Equip Sets で 表示/非表示 する装備の組み合わせを登録できるようになりました。
        • Event設定も従来通りJSON できます。
          • EventEquipSetEnable
          • EventEquipSetDisable
          • EventEquipSetDisableAll

今後修正すること#

  • Level Design
    • イベントシステムの作成
      • https://assetstore.unity.com/packages/templates/systems/easy-event-maker-88686
        • 使い物にならなそう
      • 公式の Timeline を拡張しつつ使おう
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
    • Equip Items の表示切替時のエフェクトを追加できるようにする
  • Recollect Actor
    • Running
      • Start / End のモーション設定追加
    • Sound
      • 足音の追加
    • その他
      • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
  • Recollect Director
    • その他
      • SceneView で PostEffect の On/Off を切り替えられるようにする
      • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
      • コマ止めのアニメ表現の実現方法を調べる
  • Shader
    • キャラクターにカラーパレット設定を追加
  • Config
    • キーコンフィグ画面の作成
