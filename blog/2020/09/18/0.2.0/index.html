Recollect 0.2.0

September 18, 2020 · One min read
kumak1
Director, Chief Engineer

ダウンロード#

Recollect.2020.0918.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • キャラクターの反転移動について
      • ロックオンしていない場合、背後方向に移動する際にターンアニメーションを再生する。
      • ロックオンしている場合、左<->右、前<->後 もターンアニメーションを再生する。
      • ロックオンしている場合、反転移動時に移動速度を下がるようになりました。
    • Event EventLockOnBreak を追加し、ロックオン解除を任意で行えるようにしました。
      • 設定値の詳細は ドキュメント を確認ください
    • Event EventRotate を追加し、アニメ再生中に指定角度回転できるようにしました。
      • 設定値の詳細は ドキュメント を確認ください
  • その他
    • Rewired を 1.1.36.1 にバージョンアップしました。

今後修正すること#

  • Recollect Actor
    • Humanoid 対応
      • Humanoid モーションを使えるよう調査・動くようにする
    • Offence Animation Setting の 1st に Directional を追加する。（ 2nd 以降のものと同等）
    • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
    • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する。
