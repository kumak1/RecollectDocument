Recollect 0.1.0

September 6, 2020 · One min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2020.0906.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • アニメーション合成処理をリファクタリングし、処理負荷の軽減と、合成処理を追加しやすい形に修正しました。
    • PlugableAnimeMove のパラメータにロックオンした際のモーションを設定できるように項目を追加しました。
  • Recollect Director
    • LockOn Type を追加しました。ロックオンを継続する状態を player enemy それぞれ指定できるようになりました。
  • その他
    • Rewired を 1.1.36.0 にバージョンアップしました。

今後修正すること#

  • Recollect Actor
    • Offence Animation Setting の 1st に Directional を追加する。（ 2nd 以降のものと同等）
    • 転倒 状態を追加する。（現在は 通常 戦闘 のみ）
    • キャラクターの反転移動について
      • ロックオンしていない場合、背後方向に移動する際にターンアニメーションを再生する。
      • ロックオンしている場合、左<->右、前<->後 もターンアニメーションを再生する。
      • 反転移動時に移動速度を下げる
    • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する。
    • Humanoid 対応
    • ロックオン解除のEvent追加
