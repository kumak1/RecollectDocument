Recollect 0.3.5

June 26, 2021 · One min read
kumak1
Director, Chief Engineer

ダウンロード#

  • Recollect.2021.0626.1.script.unitypackage
  • Recollect.2021.0626.1.move.unitypackage

変更箇所#

  • Recollect Actor
    • Walk
      • 歩き終わりモーションが設定できるようになりました。
      • Settings > Move > Stop > Detect Time で、Stopモーションの起動に最低限必要なボタン押下時間を設定できます

      • Turn後に歩けるようになりました

      • Speed の Turn に Scout 距離（Short, Middle, Long）に応じた設定をできるようにしました
  • Shader
    • ColorMap の Enum EIGHT -> SEVEN に変更しました
  • その他
    • FImpossible Creations のアセットを更新しました

Recollect 0.2.13

February 13, 2021 · One min read
kumak1
Director, Chief Engineer

ダウンロード#

Recollect.2021.0213.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Mask
      • SkinSwitch の Mask テクスチャが正しく動作していない問題を解決しました
    • BlendShape
      • Overwrite Object > BlendShapes で Skin Switch Param の BlendShape を上書きできるようになりました。
        • 後述の共通で利用する Equip Items の Blend Shape の上書きなどに利用してください。
    • Equip Items
      • SkinSwitch を従来のものと別途設定できるようになりました。
      • Equip Items の SkinSwitch では Default Equip Sets のチェックがされたものだけ表示されます。
      • InOut Equip Sets で 表示/非表示 する装備の組み合わせを登録できるようになりました。
        • Event設定も従来通りJSON できます。
          • EventEquipSetEnable
          • EventEquipSetDisable
          • EventEquipSetDisableAll
