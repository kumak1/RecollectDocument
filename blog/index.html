Recollect 0.3.2

May 7, 2021 · One min read

kumak1

Director, Chief Engineer

ダウンロード#

  • Recollect.2021.0507.1.script.unitypackage
  • Recollect.2021.0507.1.offence.unitypackage

変更箇所#

  • Recollect Actor
    • Lock On
      • Lock On Move の場合、前回入力とは真逆の方向に移動する場合は速度が落ちるようになりました
        • Reverse Dir Attenuation > Tolerance などで調整できます
    • Turn後の Dodge にも Basic か専用設定を選択できるようになりました
    • 通常時とバトル時で設定を変更できるように、Dodge1, Dodge2 を設定できるようになりました
    • Turn後の攻撃で移動パラメータがリセットされていない問題を解決しました
  • その他
    • Rewired を 1.1.39.1 にバージョンアップしました
    • Tail Animator を 2.0.5 にバージョンアップしました
    • Spine Animator を 2.0.0 にバージョンアップしました
    • Look Animator を 2.0.1 にバージョンアップしました

Recollect 0.2.13

February 13, 2021 · One min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2021.0213.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Mask
      • SkinSwitch の Mask テクスチャが正しく動作していない問題を解決しました
    • BlendShape
      • Overwrite Object > BlendShapes で Skin Switch Param の BlendShape を上書きできるようになりました。
        • 後述の共通で利用する Equip Items の Blend Shape の上書きなどに利用してください。
    • Equip Items
      • SkinSwitch を従来のものと別途設定できるようになりました。
      • Equip Items の SkinSwitch では Default Equip Sets のチェックがされたものだけ表示されます。
      • InOut Equip Sets で 表示/非表示 する装備の組み合わせを登録できるようになりました。
        • Event設定も従来通りJSON できます。
          • EventEquipSetEnable
          • EventEquipSetDisable
          • EventEquipSetDisableAll
