Recollect 0.2.14

February 20, 2021 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2021.0213.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Equip Items
      • enable/disable 時に Materialize FX を設定できるようになりました
    • Sound
      • サウンドエンジンとして FMOD を採用しました。
      • Event EventAudio を追加し、アニメ再生中に音を再生できるようにしました。
        • 設定値の詳細は ドキュメント を確認ください。
    • Turn
      • Turn後につながる攻撃を Basic1st か AfterTurn1st どちらするか選べるようになりました
      • Turn の途中で Dudge ができるようにする
    • Lock On
      • 左右後の移動モーション
  • Recollect Director
    • 背景をうつすと処理負荷が高い原因を探る
  • Level Design
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。

Recollect 0.2.13

February 13, 2021 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2021.0213.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • Mask
      • SkinSwitch の Mask テクスチャが正しく動作していない問題を解決しました
    • BlendShape
      • Overwrite Object > BlendShapes で Skin Switch Param の BlendShape を上書きできるようになりました。
        • 後述の共通で利用する Equip Items の Blend Shape の上書きなどに利用してください。
    • Equip Items
      • SkinSwitch を従来のものと別途設定できるようになりました。
      • Equip Items の SkinSwitch では Default Equip Sets のチェックがされたものだけ表示されます。
      • InOut Equip Sets で 表示/非表示 する装備の組み合わせを登録できるようになりました。
        • Event設定も従来通りJSON できます。
          • EventEquipSetEnable
          • EventEquipSetDisable
          • EventEquipSetDisableAll

Recollect 0.2.5

October 18, 2020 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2020.1018.1.script.unitypackage

変更箇所#

  • Recollect Director
    • シーン管理の名称を Situation > Cut から Situation > Shot に変更しました
    • Follow Camera 機能を Bypass できるようになりました
    • Receive Wind 機能を Bypass できるようになりました
    • Close Range Hologram 機能を Bypass できるようになりました
    • Generate Prefab 項目を追加しました
      • Play した際、Recollect Director 配下に生成されます
    • Spawn Actors Param に Layer を指定できるようになりました
      • 3D Game Kit では、キャラクターの Layer が厳密に決められていたので設定追加
    • Lighting Param に Rendering Path 項目を追加しました
      • 現在の Recollect Shader は Forward Rendering 飲み対応しています
      • 3D Game Kit を Import すると Diferred で上書きされるので設定追加
