Recollect 0.10.1

December 14, 2022 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.1214.1.script.unitypackage

変更箇所​

  • Asset
    • Unity Registry の以下Package を利用するようになったので、 Package Manager から install してください。
      • Burst
      • TextMeshPro
  • RecollectDirector
    • respawn 後にカメラ接近したら hologramにならない問題を修正しました
    • respawn 後にのShadow meshが非表示のままな問題を修正しました
    • respawn 時、Middle Range内にPlayerがいる場合は、respawnする座標をずらすようにしました
    • respawn の設定を、Director で共通設定で持つようにしました。
      • 上書きしたい場合は、設定ファイルを個別で設定してください。
        • 設定ファイルは、Project ウィンドウで右クリックし、 Create > Recollect > Director > Spawn > Respawn をクリックします。
  • RecollectActor
    • キャラクターのHPが0になってダメージモーションを再生する際、super armor にしていましたが死体蹴りモーションができるようしました
