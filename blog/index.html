変更箇所​

  • BarricadeSetUpper
    • prefabの中で設定できるのか確認できました
  • RecollectDirector
    • spawn
      • spawn時の地面が0で ray 飛ばす
  • RecollectUiEvent
    • Equip Item Set の toggle 時の icon 切り替えできるようになりました
      • RecollectUiImageInjector の component を追加し、 RecollectUiEvent で ImageRefresh の設定をしてください。

    • 現状のPlayer情報を使って次のSituationに行く方法をまとめる
      • Save Current Player にチェックし、 Player を Use Saved すると、変更したPlayerを維持できます。
    • skirt の物理演算のリファレンスを作成
      • Rotate Limit の x軸 で内側に入らないようにし、z軸 も少し許可して左右にちょっとだけ揺れるようにすると良さそうです

    • Shader Compiler

Recollect 0.10.9

April 9, 2023 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2023.0409.1.script.unitypackage

変更箇所​

キーコンフィグにまつわる機能を追加しました

  • Config
    • RecollectUiEvent
      • Mode に KeyConfig を追加しました
        • Listen Binding : 指定アクションのキー設定を追加変更します。UIをクリックした後に、キー入力をすると設定の追加・変更されます。
        • Save : 現状の設定を保存します
        • Load : 保存された設定をロードします
        • Reset : 設定を初期状態に戻します
    • RecollectTextInjector
      • Injection From に KeyConfig にまつわる設定値を追加しました
    • RecollectDirector
      • AutoLoad KeyConfig を追加しました
        • Editor の再生時に保存済みの設定を読み込みます
