Recollect 0.5.1

January 26, 2022 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0126.1.unitypackage

変更箇所​

  • Recollect Actor
    • BoneScale に Rotate の設定を追加しました
      • Breast_L など Blender の Animation がないboneだと常に加算し続ける挙動になっていたので修正しました
    • Equip Items
      • Description を追加しました
      • Equip Items 内の Skin Switch を配列にしました
      • Slotに項目を追加
        • LR なしを追加しました
        • Gear 追加しました
    • 廃止予定の項目を削除しました
      • Equip Items > Destructible > SkinSwitch (deprecated) を削除しました
      • BoneScale (deprecated) を削除しました
      • BoneParam (deprecated) を削除しました
  • Recollect Director
    • コマ止めのアニメ表現を実装しました
    • ScreenSetting > Screen Param > Situation を StriptableObject をやめました
    • カメラを回転させてキャラクターの背面に回った際、カメラがY軸上方に移動する現象を解決しました
      • RotateLimit にちょっと遊びを持たせたら解決しました
  • Menu
    • メニューのアセットを追加しました

Recollect 0.4.9

November 14, 2021 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2021.1114.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • Profile
      • Equip Item Set の項目を設定ファイルで指定できるようになりました
        • Bone Scale
        • Bone Param
        • Tail Animator
      • Equip Item Set で同名Meshを複数設定している場合は上書きされるバグを修正しました
      • SkinSwitch(deprecated) を設定しない場合は全GameObjectを表示していましたが、全GameObjectを非表示に変更しました
  • Recollect Director
    • menu の OSS
  • Shader
    • VertexMagic を試す
      • 結論 : 使えなさそう
        • Material 全体にエフェクトがかかる
          • 早く動いた箇所（指先や剣先など）を強く伸ばすことができない
        • 直線でしか伸ばせない
  • そのほか
    • MKGlow 4.4.10 にアップデートしました
    • UnityScreenNavigator の導入を決定しました
      • https://github.com/Haruma-K/UnityScreenNavigator/blob/master/README_JA.md
