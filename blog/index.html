Recollect 0.10.9

2023年4月9日 · 約3分
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2023.0409.1.script.unitypackage

変更箇所​

キーコンフィグにまつわる機能を追加しました

  • Config
    • RecollectUiEvent
      • Mode に KeyConfig を追加しました
        • Listen Binding : 指定アクションのキー設定を追加変更します。UIをクリックした後に、キー入力をすると設定の追加・変更されます。
        • Save : 現状の設定を保存します
        • Load : 保存された設定をロードします
        • Reset : 設定を初期状態に戻します
    • RecollectTextInjector
      • Injection From に KeyConfig にまつわる設定値を追加しました
    • RecollectDirector
      • AutoLoad KeyConfig を追加しました
        • Editor の再生時に保存済みの設定を読み込みます
