Director, Chief Engineer

ダウンロード​

  • Recollect.2021.1114.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • Profile

        • Bone Param
        • Tail Animator
      • Equip Item Set で同名Meshを複数設定している場合は上書きされるバグを修正しました

    • menu の OSS
  • Shader
    • VertexMagic を試す
      • 結論 : 使えなさそう
        • Material 全体にエフェクトがかかる
          • 早く動いた箇所（指先や剣先など）を強く伸ばすことができない
        • 直線でしか伸ばせない
  • そのほか
    • MKGlow 4.4.10 にアップデートしました
    • UnityScreenNavigator の導入を決定しました
      • https://github.com/Haruma-K/UnityScreenNavigator/blob/master/README_JA.md

Recollect 0.2.3

September 22, 2020 · 2 min read
kumak1
Director, Chief Engineer

ダウンロード​

Recollect.2020.0922.1.script.unitypackage

変更箇所​

  • Shader
    • Color Map に8色モードを追加しました。
      • Multiply のみ利用可能です。
      • Color Map に利用できる色の種類は下記8種類
        • Red (R: 255, G: 0, B: 0, A: 255)
        • Green (R: 0, G: 255, B: 0, A: 255)
        • Blue (R: 0, G: 0, B: 255, A: 255)
        • Cyan (R: 0, G: 255, B: 255, A: 255)
        • Magenta (R: 255, G: 0, B: 255, A: 255)
        • Yellow (R: 255, G: 255, B: 0, A: 255)
        • Black (R: 0, G: 0, B: 0, A: 255)
        • White (R: 255, G: 255, B: 255, A: 255)
      • Color Map の対象外にするには A: 0 にしてください。
      • 色情報が真逆の色が隣にあるとアーティファクトが発生するので利用しないでください。
        • Red (R: 255, G: 0, B: 0) と Cyan (R: 0, G: 255, B: 255)
        • Green (R: 0, G: 255, B: 0) と Magenta (R: 255, G: 0, B: 255)
        • Blue (R: 0, G: 0, B: 255) と Yellow (R: 255, G: 255, B: 0)
        • Black (R: 0, G: 0, B: 0) と White (R: 255, G: 255, B: 255)
  • その他
    • InControl を 1.8.3 にバージョンアップしました。
