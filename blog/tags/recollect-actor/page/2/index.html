Director, Chief Engineer

変更箇所​

  • RecollectDirector
    • 左右移動時にカメラの zoom in/out の挙動が不安定になる問題を修正しました
  • RecollectActor
    • ScoutRange の Offset を指定、中心地を変更できる様になりました
    • DamageEffect の OffsetType を、 None Specify Attacker Hit Offset から選択できるようになりました
      • None : 攻撃が当たった箇所で Effect を表示します
      • Specify : 上記に加え、位置をズラしたい場合に指定します
      • Attacker Hit Offset : 攻撃をしているキャラクターの Hit Offset の位置で Effect を表示します
