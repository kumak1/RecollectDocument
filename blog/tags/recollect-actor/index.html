Recollect 0.6.15

August 18, 2022 · 4 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0818.1.script.unitypackage

変更箇所​

  • RecollectUiEvent
    • Keep Position 項目を追加し、Player の切替時に Rotation を維持するか、Spawn Param に従うか選択できるようになりました
    • Equip の optional を UI から変更したい
      • Recollect Actor の Debug > Enable Equip Item Sets のチェックを外すと動作します。チェックの状態にご注意ください
  • RecollectActor
    • Normal <-> Battle の状態遷移時にキー入力を受付しないように修正しました
    • Scout (距離の判定) を2frameで分散処理していたのを、1frame内で完了するように変更しました
    • 服装切替時のエフェクト再生できるようになりました
      • RecollectUIEvent の Effect > Equip Change にチェックがあることを確認します
      • EquipSet の SkinSwitch > EquipSetting > Effect に指定されたEffectを再生します
      • 注意 : UI をクリックして切り替わった装備のEffectだけ再生されます
    • EquipSet にアイコン登録できるようになりました
      • Equip Item 毎に Icon を指定できます
      • RecollectUiImageInjector を Add Component します

Recollect 0.6.14

August 8, 2022 · 4 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0808.1.script.unitypackage

変更箇所​

  • RecollectDirector
    • 左右移動時にカメラの zoom in/out の挙動が不安定になる問題を修正しました
  • RecollectActor
    • ScoutRange の Offset を指定、中心地を変更できる様になりました
    • DamageEffect の OffsetType を、 None Specify Attacker Hit Offset から選択できるようになりました

      • Attacker Hit Offset : 攻撃をしているキャラクターの Hit Offset の位置で Effect を表示します
