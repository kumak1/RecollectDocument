Recollect 0.2.5

October 18, 2020 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2020.1018.1.script.unitypackage

変更箇所#

  • Recollect Director
    • シーン管理の名称を Situation > Cut から Situation > Shot に変更しました
    • Follow Camera 機能を Bypass できるようになりました
    • Receive Wind 機能を Bypass できるようになりました
    • Close Range Hologram 機能を Bypass できるようになりました
    • Generate Prefab 項目を追加しました
      • Play した際、Recollect Director 配下に生成されます
    • Spawn Actors Param に Layer を指定できるようになりました
      • 3D Game Kit では、キャラクターの Layer が厳密に決められていたので設定追加
    • Lighting Param に Rendering Path 項目を追加しました
      • 現在の Recollect Shader は Forward Rendering 飲み対応しています
      • 3D Game Kit を Import すると Diferred で上書きされるので設定追加

Recollect 0.1.0

September 6, 2020 · 1 min read

kumak1

Director, Chief Engineer

ダウンロード#

Recollect.2020.0906.1.script.unitypackage

変更箇所#

  • Recollect Actor
    • アニメーション合成処理をリファクタリングし、処理負荷の軽減と、合成処理を追加しやすい形に修正しました。
    • PlugableAnimeMove のパラメータにロックオンした際のモーションを設定できるように項目を追加しました。
  • Recollect Director
    • LockOn Type を追加しました。ロックオンを継続する状態を player enemy それぞれ指定できるようになりました。
  • その他
    • Rewired を 1.1.36.0 にバージョンアップしました。
