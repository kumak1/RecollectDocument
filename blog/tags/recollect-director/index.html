Recollect 0.10.9

April 9, 2023 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2023.0409.1.script.unitypackage

変更箇所​

キーコンフィグにまつわる機能を追加しました

  • Config
    • RecollectUiEvent
      • Mode に KeyConfig を追加しました
        • Listen Binding : 指定アクションのキー設定を追加変更します。UIをクリックした後に、キー入力をすると設定の追加・変更されます。
        • Save : 現状の設定を保存します
        • Load : 保存された設定をロードします
        • Reset : 設定を初期状態に戻します
    • RecollectTextInjector
      • Injection From に KeyConfig にまつわる設定値を追加しました
    • RecollectDirector
      • AutoLoad KeyConfig を追加しました
        • Editor の再生時に保存済みの設定を読み込みます

Recollect 0.10.1

December 14, 2022 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.1214.1.script.unitypackage

変更箇所​

  • Asset
    • Unity Registry の以下Package を利用するようになったので、 Package Manager から install してください。
      • Burst
      • TextMeshPro
  • RecollectDirector
    • respawn 後にカメラ接近したら hologramにならない問題を修正しました
    • respawn 後にのShadow meshが非表示のままな問題を修正しました

  • RecollectActor
    • キャラクターのHPが0になってダメージモーションを再生する際、super armor にしていましたが死体蹴りモーションができるようしました
