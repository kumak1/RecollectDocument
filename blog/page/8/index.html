Recollect 0.3.0

2021年3月20日 · 約2分
kumak1
Director, Chief Engineer

ダウンロード​

Recollect.2021.0321.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • Turn
      • PlugableAnimeOffence に設定値を追加しました
      • Turn後につながる攻撃を Basic1st か AfterTurn1st どちらするか選べるようになりました
      • Turn の途中で Dudge ができるようになりました。
    • Equip Items
      • enable/disable 時に Materialize FX を設定できるようになりました
    • Sound
      • サウンドエンジンとして FMOD を採用しました。
      • Event EventAudio を追加し、アニメ再生中に音を再生できるようにしました。
        • 設定値の詳細は ドキュメント を確認ください。

Recollect 0.2.13

2021年2月13日 · 約3分
kumak1
Director, Chief Engineer

ダウンロード​

Recollect.2021.0213.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • Mask
      • SkinSwitch の Mask テクスチャが正しく動作していない問題を解決しました
    • BlendShape
      • Overwrite Object > BlendShapes で Skin Switch Param の BlendShape を上書きできるようになりました。
        • 後述の共通で利用する Equip Items の Blend Shape の上書きなどに利用してください。
    • Equip Items
      • SkinSwitch を従来のものと別途設定できるようになりました。
      • Equip Items の SkinSwitch では Default Equip Sets のチェックがされたものだけ表示されます。
      • InOut Equip Sets で 表示/非表示 する装備の組み合わせを登録できるようになりました。
        • Event設定も従来通りJSON できます。
          • EventEquipSetEnable
          • EventEquipSetDisable
          • EventEquipSetDisableAll

Recollect.2020.1220.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • SkinSwitch
      • 閉じる 入れ替え ができるようになりました（Unity 2020.2 の Reorderable に対応）
    • Turn
      • アニメーションの Blend In/Out が動作していない問題を修正しました
