Recollect 0.6.10

June 22, 2022 · 4 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0622.1.script.unitypackage

変更箇所​

  • FBX Importer
    • EventAttack
      • Aim を即時に方向転換するか設定できるようにしました
      • FBX Animation Param で設定する場合
        • EventAttack の設定項目で Immediate Turn のチェックを入れます
      • JSON で設定する場合
        • EventAttack の設定項目で UseAimImmediateTurn に true を指定します

  • RecollectUiEvent
    • EquipSet > Toggle の順序を name で sort をするようにしました

Recollect 0.6.6

May 27, 2022 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0527.1.script.unitypackage

変更箇所​

  • Recollect Actor
    • 眼帯など、一部だけ変える装備の設定項目 is Optional を追加しました
  • RecollectUiEvent
    • Player > Overwrite Param > EquipSet に KeepOptional の設定値を追加しました。
      • true の場合、 is Optional の表示・非表示は維持されます。
      • false の場合は is Optional が全て非表示になります。
    • EquipSet に Optional を追加しました
      • EquipSet のうち Optional だけの操作をおこないます
  • RecollectUiTextInjector
    • EquipSet の Toggle で変えられる数をtextに出せるようになりました
