Recollect 0.5.6

March 13, 2022 · 3 min read
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0313.1.script.unitypackage

変更箇所​

  • RecollectDirector
    • Ocean を RecollectDirector から呼び出す際に発生するエラーを修正しました
    • Inspector の表示順序を更新しました
      • Situation > Stage の Camera の表示を最下段に移動しました
      • Situation > Construction と Situation > Spawn の位置を入れ替えました
    • カメラを回転させたら、アクターに紐づく小物（靴など）がカメラと衝突して？非表示になる問題を修正しました

今後修正すること​

  • Recollect Actor
    • Guard 転倒 状態を追加する（現在は 通常 戦闘 のみ）
      • Guard 状態を追加する
      • 転倒 状態を追加する
    • 真っ二つ
      • https://www.youtube.com/watch?v=i--UUeGTMvo
  • Recollect Director
    • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
    • ローディングのシークバーが欲しい
    • キャラクター切替（Q, E キー）
  • Level Design
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Shader
    • キャラクターにカラーパレット設定を追加
    • プロシージャルでテクスチャ作れるようにする
  • Sound
    • FMODに変わる音再生システムの構築
  • Config
    • キーコンフィグ画面の作成
    • WebGL で MouseOverしたときにカーソルを変えられる？
  • Screen Shot
    • パネルの位置保存
  • WebGL Build
    • カーソルが変えられるか調査
  • Unity Editor が Fatal Error で落ちる問題の調査
    • 現状確認
      • 再現方法 : Situation Cut の変更時に Unity Editor が落ちる
      • 条件を絞る : Enable Process で Spawn を off にすると落ちない
      • BlendShape の設定適用関数 SetBlendShapeWeight() でエラーが頻発していそう
  • Menu
    • メニュープロトタイプ作成
    • カメラ切り替え設定の追加
  • カメラを回転させたら、アクターに紐づく小物（靴など）がカメラと衝突して？非表示になる
  • Unity Editor が Fatal Error で落ちる問題の調査
    • 現状確認
      • 再現方法 : Situation Cut の変更時に Unity Editor が落ちる
      • 条件を絞る : Enable Process で Spawn を off にすると落ちない
      • BlendShape の設定適用関数 SetBlendShapeWeight() でエラーが頻発していそう
      • Spawn Param で同じ FBX の Actor が複数存在するとエラーになる
