Recollect 0.6.17

2022年8月28日 · 約3分
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2022.0903.1.script.unitypackage

変更箇所​

  • WebGL
    • WebGL で MouseOverしたときにカーソルが変えられるようになりました
    • RecollectDirector の Hover でマウスオーバー時のカーソル画像を指定できるようになりました
    • RecollectUiEvent の Cursor > Hover でマウスオーバーでカーソル画像を変更するか指定できる様になりました

今後修正すること​

  • Menu
    • https://github.com/Haruma-K/uPalette を導入し、文字色などを一括管理できるように
    • 指定prefix の EquipSet がない場合、UI を 非表示にする Component の作成
  • RecollectUiEvent
  • RecollectActor
    • Guard 転倒 状態を追加する（現在は 通常 戦闘 のみ）
      • Guard 状態を追加する
      • 転倒 状態を追加する
    • 真っ二つ
      • https://www.youtube.com/watch?v=i--UUeGTMvo
    • EquipSet に FootStepCollider 作成
    • アニメーションを逆再生できないか確認
    • 攻撃の回数を増やす
  • RecollectDirector
    • 攻撃などのエフェクトの初回起動時の動作が重いので、オブジェクトプールを導入する
    • キャラクター切替（Q, E キー）
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Shader
    • キャラクターにカラーパレット設定を追加
    • Bone Weight の拡大縮小の状態によって、normal map の適用状態を変えたい
      • https://twitter.com/_Uraki/status/1533482916402978816 こんなん
  • Sound
    • FMODに変わる音再生システムの構築
  • Config
    • キーコンフィグ画面の作成
  • Screen Shot
    • パネルの位置保存
  • etc
    • EquipSet の Array の名称が Elemental じゃなくて 装備名になるといいな
    • Layer を再現するスクリプトをつくる
