Recollect 0.11.11

2024年3月3日 · 約3分
kumak1
Director, Chief Engineer

ダウンロード​

  • Recollect.2024.0303.1.script.unitypackage
  • Recollect.2024.0303.1.incontrol.unitypackage

変更箇所​

  • UI
    • VerticalText を追加しました
      • Text Componentの文字列を縦書きに変更できます
      • 設定ファイルは　Create > Recollect > UI > Text > VerticalText から作成できます
      • Rotate を -90 にして利用してください

今後修正すること​

  • 一つのEquipSetでない時アクタの変更の時ホログラムが出ない
  • 画面切り替えのとき
    • シーン切り替えの演出・パーリンノイズとか欲しい
  • AI (Behavior Tree)
    • プレイヤーとカメラの間は避けるように移動したい
    • add Conditionals
      • duration
    • add Action
      • ActorInputMove
      • LockOnActor
  • RecollectActor
    • Guard 転倒 状態を追加する（現在は 通常 戦闘 のみ）
      • Guard 状態を追加する
      • 転倒 状態を追加する
    • 真っ二つ
      • https://www.youtube.com/watch?v=i--UUeGTMvo
    • EquipSet に FootStepCollider 作成
  • RecollectDirector
    • キャラクター切替（Q, E キー）
    • カットシーンのイベントの作成
      • Asset はダメそうなので、公式の Timeline を拡張する。
  • Effect
    • Blow Hit で壁や敵に当たった時のエフェクトを追加できるようにする
  • Sound
    • FMODに変わる音再生システムの構築
  • Config
    • キーコンフィグ画面の作成
  • etc
    • Layer を再現するスクリプトをつくる
    • Shader
      • Bone Weight の拡大縮小の状態によって、normal map の適用状態を変えたい
        • こんなん
      • キャラクターにカラーパレット設定を追加
    • アニメーションを逆再生できないか確認
  • FBX Importer
    • JSON を用いた自動設定機能を削除
    • FbxAnimationParam のみの対応になるので、ご注意ください
      • JSONファイルを右クリック（複数選択可能）し、 Recollect > FbxAnimation > Convert > JsonToAsset を選択して、JSONファイルから FbxAnimationParam を生成してください
